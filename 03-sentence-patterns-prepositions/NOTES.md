## Part 1 — The 6 Core Sentence Patterns

Yeh **6 patterns** master karo. **80% interview cover** ho jayega.

### Pattern 1: Current Task

**`I am working on...`**

| Hindi | English |
|---|---|
| Main payment app par kaam kar raha hu | **I am working on** a payment app |
| Main pipeline issue fix kar raha hu | **I am working on** pipeline issues |

### Pattern 2: Tools

**`I work with... / I use...`**

| Hindi | English |
|---|---|
| Main Jenkins use karta hu | **I work with** Jenkins |
| Hum Azure use karte hain | **We use** Azure |

### Pattern 3: Role / Responsibility

**`My responsibility is to... / I am responsible for...`**

| Hindi | English |
|---|---|
| Mera kaam deployments handle karna hai | **I am responsible for** handling deployments |
| Mera role infrastructure manage karna hai | **My responsibility is to** manage infrastructure |

### Pattern 4: Team

**`We have ... in our team`**

| Hindi | English |
|---|---|
| Hamari team mein 10 log hain | **We have** 10 people **in our team** |
| 3 DevOps engineers hain | **We have** 3 DevOps engineers |

### Pattern 5: Experience

**`I have been ... for / since...`**

| Hindi | English |
|---|---|
| 2 saal se kaam kar raha hu | **I have been working for** 2 years |
| Pichle 6 mahine se Terraform use kar raha hu | **I have been using** Terraform **for the last** 6 months |

### Pattern 6: Outcome / Benefit

**`It helps us to... / This allows us to...`**

| Hindi | English |
|---|---|
| Isse releases fast hoti hain | **It helps us** release faster |
| Isse automation hoti hai | **This allows us to** automate everything |

## Part 2 — Important Prepositions

Sahi preposition sabse zyada confusing hai Hindi speakers ke liye. Yeh 4 categories master karo.

### For Work / Company

| Preposition | Use For | Example |
|---|---|---|
| **AT** | specific company | I work **at** Google |
| **FOR** | employer | I work **for** Google |
| **IN** | field / city | I work **in** DevOps, **in** Pune |
| **WITH** | client / team | I work **with** a fintech client |
| **ON** | project / task | I work **on** a payment app |

**COMMON MISTAKE:** *"I work **in** Google"* — WRONG!
**RIGHT:** *"I work **at** Google"*

### For Time / Duration

| Preposition | Use For | Example |
|---|---|---|
| **AT** | exact time | Standup **at** 10 AM |
| **ON** | day / date | We deploy **on** Friday |
| **IN** | month / year | I joined **in** June 2023 |
| **FOR** | duration | Working **for** 2 years |
| **SINCE** | starting point | Working **since** 2022 |

**COMMON MISTAKE:** *"Working **since** 2 years"* — WRONG!
**RIGHT:** *"Working **for** 2 years"* OR *"Working **since** 2022"*

### For Tools / Platforms

| Preposition | Use For | Example |
|---|---|---|
| **WITH** | tools alongside | I work **with** Jenkins |
| **USING** | active tool use | I deploy **using** Terraform |
| **ON** | platform / environment | Deploy **on** Azure |
| **VIA** | method / route | Connect **via** SSH |
| **THROUGH** | via a process | Goes **through** the pipeline |

### For Deployment / Movement

| Preposition | Use For | Example |
|---|---|---|
| **TO** | destination | Deploy **to** production |
| **FROM** | source | Pull code **from** GitHub |
| **INTO** | going inside | Push **into** main branch |
| **ACROSS** | multiple things | Consistent **across** environments |
| **BETWEEN** | 2 things | Flow **between** dev and prod |

## Common Mistakes Table

| Wrong | Right |
|---|---|
| I work **in** Google | I work **at** Google / I work **for** Google |
| I work **since** 3 years | I have been working **for** 3 years / **since** 2022 |
| I am working **at** a project | I am working **on** a project |
| Deploy **in** production | Deploy **to** production |
| I deploy **in** AWS | I deploy **on** AWS |
| I work **with** DevOps | I work **in** DevOps |
| Discuss **on** the issue | Discuss the issue / Discuss **about** the issue |

## The Master Sentence (YAAD KARO!)

Yeh ek sentence **rate ki tarah yaad** kar lo. Isme **8 prepositions** ek hi line mein hain:

> *"I work **AT** XYZ company **AS** a DevOps Engineer **IN** the fintech domain. I have been working **WITH** Terraform **FOR** 2 years. I work **ON** infrastructure automation **USING** Azure DevOps and deploy our applications **TO** production **ACROSS** multiple environments."*

Iss ek sentence mein:

- **at** → company
- **as** → role
- **in** → field
- **with** → tool
- **for** → duration
- **on** → task
- **using** → active tool
- **to** → destination
- **across** → multiple
